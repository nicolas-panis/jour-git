Tableaux de commandes Git

Commandes Description
git config L’une des commandes git les plus utilisées est git config. On l’utilise pour configurer les préférences de l’utilisateur : son mail, l’algorithme utilisé pour diff, le nom d’utilisateur et le format de fichier etc.
git init Cette commande est utilisée pour créer un nouveau dépôt GIT
git clone La commande git clone est utilisée pour la vérification des dépôts.
git commit La commande git commit permet de valider les modifications apportées au HEAD. Notez que tout commit ne se fera pas dans le dépôt distant.
git status La commande git status affiche la liste des fichiers modifiés ainsi que les fichiers qui doivent encore être ajoutés ou validés.
git checkout La commande git checkout peut être utilisée pour créer des branches ou pour basculer entre elles.
git remote Cette commande remote permet à un utilisateur de se connecter à un dépôt distant.
git branch La commande git branch peut être utilisée pour répertorier, créer ou supprimer des branches.
git pull Pour fusionner toutes les modifications présentes sur le dépôt distant dans le répertoire de travail local, la commande pull est utilisée
git merge La commande git merge est utilisée pour fusionner une branche dans la branche active.
git diff La commande git diff permet de lister les conflits.
git tag Le marquage est utilisé pour marquer des commits spécifiques avec des poignées simples.
git log L’ exécution de cette commande génère le log d’une branche.
git reset Pour réinitialiser l’index et le répertoire de travail à l’état du dernier commit, la commande git reset est utilisée
Git rm Git rm peut être utilisé pour supprimer des fichiers de l’index et du répertoire de travail.
git stash L’une des moins connues, git stash aide à enregistrer les changements qui ne doivent pas être commit immédiatement. C’est un commit temporaire.
git show Pour afficher des informations sur tout fichier git, utilisez la commande git show .
git fetch Git fetch permet à un utilisateur d’extraire tous les fichiers du dépôt distant qui ne sont pas actuellement dans le répertoire de travail local.
gitk Gitk est l’interface graphique du dépôt local.
git instaweb Avec la commande git instaweb , un serveur Web peut être exécuté par interface avec le dépôt local. Qui redirige directement vers un serveur web.
git gc Pour optimiser le dépôt en supprimant les fichiers inutiles et les optimiser
git archive La commande git archive permet à un utilisateur de créer un fichier zip ou tar contenant les composants d’un arbre du dépôt.
git rebase La commande git rebase est utilisée pour la réapplication des commits sur une autre branche.
